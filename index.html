Gambar Persegi

A. Luas Persegi

Rumus untuk menghitung luas persegi adalah:

L = S x S

Keterangan:

L = Luas

S = Sisi

Hitung Luas Persegi


B. Keliling Persegi

Rumus untuk menghitung keliling persegi adalah:

K = 4 x S

Keterangan:

K = Keliling

S = Sisi

Persegi Panjang

Gambar Persegi Panjang

A. Luas Persegi Panjang

Rumus untuk menghitung luas persegi panjang adalah:

L = P x l

Keterangan:

L = Luas

P = Panjang

l = Lebar

Hitung Luas Persegi Panjang


B. Keliling Persegi Panjang

Rumus untuk menghitung keliling persegi panjang adalah:

K = 2 x (P + l)

Keterangan:

K = Keliling

P = Panjang

l = Lebar

Hitung Keliling Persegi Panjang
